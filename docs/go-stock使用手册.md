## 软件简介

go-stock 是一款基于大语言模型的 AI 赋能股票分析工具，支持A股、港股、美股的市场数据分析。

**核心特性：**

- AI 智能分析股票走势
- K 线技术指标分析
- 市场情绪分析
- 智能选股
- 实时行情监控
- 资金流向追踪
- 龙虎榜数据
- 财经资讯聚合

**技术架构：**

- 后端：Go + Wails
- 前端：Vue 3 + NaiveUI
- AI：大模型对话分析

## 安装与启动

## 股票功能

### 个股详情

在股票页面，可以查看个股的详细信息：

**行情数据：**

- 实时价格
- 涨跌幅
- 最高价/最低价
- 成交量/成交额
- 市值/流通市值
- 市盈率/市净率

**分时图：**

- 显示股票当日分时走势
- 支持鼠标滚轮缩放
- 显示成交量柱状图

**K 线图：**

- 日线、周线、月线切换
- 支持右键菜单查看更多周期
- 显示 MA5、MA10、MA20 等均线
- 支持布林带、MACD、KDJ 等技术指标

**关注功能：**

- 点击「关注」按钮将股票添加到关注列表
- 关注列表可在左侧边栏查看
- 支持分组管理

### K 线周期设置

在 K 线图右上角可以设置：

- **K 线天数**：查看最近 N 天的 K 线数据
- **技术指标**：切换显示不同技术指标

### 股票预警

设置涨跌幅预警：

- 当股票涨跌达到设定值时，会收到系统通知
- 预警价格可自定义设置

## 市场行情

### 全球指数

在市场页面顶部，显示全球主要指数：

- **A股**：上证指数、深证成指、创业板指、科创50
- **美股**：道琼斯、纳斯达克、标普500
- **港股**：恒生指数、恒生国企指数
- **欧洲**：英国富时、德国DAX、法国CAC40
- **亚洲**：日经225、韩国综合等

指数数据每3秒自动刷新。

### 热门股票

显示当前市场最热门的股票榜单，包括：

- 股票名称和代码
- 涨跌幅
- 当前价格
- 热度值
- 排名变化

鼠标悬停可弹出 K 线图查看详情。

### 行业资金排名

查看各行业的资金流向情况：

| 指标   | 说明          |
| ---- | ----------- |
| 板块名称 | 行业/板块名称     |
| 涨跌幅  | 板块整体涨跌      |
| 流入资金 | 资金净流入额      |
| 流出资金 | 资金净流出额      |
| 净流入  | 净流入金额       |
| 领涨股  | 该板块中涨幅最大的股票 |

支持按净流入、净流出排序。

### 龙虎榜

龙虎榜展示交易所公布的异动股票数据：

**查看方式：**

1. 选择查询日期
2. 可按上榜原因筛选
3. 查看个股的龙虎榜详情

**龙虎榜数据包括：**

- 股票代码和名称
- 上榜原因
- 营业部买卖数据
- 机构买卖情况

支持查看历史龙虎榜数据。

### 股票板块分布

以树状图/热力图形式展示股票分布情况，直观了解市场结构。

## AI智能分析

### AI 分析个股

在股票详情页面，可以对当前股票进行 AI 分析：

1. 打开股票详情
2. 点击「AI分析」按钮
3. 系统会自动收集相关资讯
4. AI 分析并生成分析报告

**分析内容：**

- 近期走势分析
- 技术面分析
- 消息面分析
- AI 给出分析结论和建议

### AI 思考模式

开启思考模式后，AI 会展示推理过程：

- 显示 AI 的分析思路
- 逐步推导结论
- 更加透明的决策过程

### 多轮对话

AI 分析完成后，可以继续提问：

- 针对分析结果追问
- 询问具体数据
- 获取投资建议

### 保存分析结果

分析结果可以：

- 保存为 Markdown 文件
- 导出为 Word 文档
- 保存分析历史记录

### AI 配置

在设置中配置 AI 模型：

**支持的模型平台：**

- OpenAI (GPT 系列)
- Ollama (本地大模型)
- LMStudio (本地大模型)
- DeepSeek
- 硅基流动
- 其他兼容 OpenAI API 格式的平台

**配置参数：**

- API 地址 (Base URL)
- API Key
- 模型名称
- Temperature (温度参数)
- 最大Token数
- 请求超时时间
- HTTP 代理设置

## AI智能体

AI智能体以悬浮助手形式呈现，可在任意页面随时调出使用。

### 悬浮球触发

在页面右侧边缘可见悬浮球按钮：

- 非 AI 助手页面时显示
- 后台有分析任务时显示忙碌标记
- 点击悬浮球展开 AI 助手抽屉面板
- 抽屉从右侧滑入，带遮罩层

### 抽屉界面

**顶部操作栏：**

- 新对话：重置对话，开始新话题
- 分享到社区：分享 AI 回复到社区
- 关闭：收起抽屉

### 对话消息

**消息展示：**

- Markdown 格式渲染
- AI 回复支持展开/收起（超过200字符自动折叠）
- 显示 AI 推理过程
- 显示消息时间戳
- 显示使用的模型名称

**消息操作（AI 回复下方）：**

- 展开/收起
- 复制内容
- 导出为图片
- 分享到社区

### 底部功能栏

**模型选择：**

- 下拉选择已配置的 AI 模型
- 支持 OpenAI、DeepSeek、Ollama、LMStudio 等

**提示词模板：**

- 系统提示词：选择预设的系统角色
- 用户提示词：选择预设的用户模板，选择后自动填充输入框

**思考模式：**

- 开关控制
- 开启后 AI 会展示完整推理过程

**记忆模式：**

- 开关控制
- 开启后可设置记忆条数（5/10/20/30/50条）
- AI 会记住最近 N 条对话内容
- 方便连续对话时保持上下文

### 发送消息

- 输入框输入内容
- 按回车键发送
- 支持 Markdown 格式
- 发送按钮或中断按钮切换
- 流式输出显示思考中状态

### 会话持久化

- 自动保存对话会话
- 重启后可加载历史会话
- 支持查看和继续历史对话

## AI智能选股

### 功能入口

在「研究」页面，选择「股票信息筛选」或「热门选股策略」。

### AI 选股方式

**方式一：自然语言选股**

- 输入选股要求，如："市值大于100亿，PE低于20的股票"
- AI 解析需求
- 返回符合条件的股票列表

**方式二：指标条件选股**
使用预设的选股指标条件：

- 市值指标
- 估值指标
- 成长指标
- 技术指标

### 选股结果

选股结果以表格展示：

- 股票代码和名称
- 相关指标数值
- 支持排序和筛选
- 点击可查看 K 线详情

### 关注股票

在选股结果中点击「关注」按钮，可将股票加入关注列表。

## 交易记录管理

### 添加交易记录

点击「添加交易」按钮，记录买入/卖出：

| 字段   | 说明        |
| ---- | --------- |
| 股票代码 | 输入或搜索股票   |
| 股票名称 | 自动填充      |
| 方向   | 买入/卖出     |
| 价格   | 成交价格      |
| 数量   | 成交数量      |
| 交易时间 | 成交时间      |
| 手续费  | 交易费用      |
| 止损价  | 止损价格      |
| 止盈价  | 止盈价格      |
| 交易原因 | 记录买入/卖出理由 |
| 持仓心态 | 记录当时心态    |

### 交易记录列表

查看历史交易记录：

- 分页展示
- 支持按日期范围筛选
- 支持按方向（买入/卖出）筛选
- 支持关键词搜索

### 统计分析

查看交易统计数据：

- 本月买入/卖出次数
- 本月累计交易金额
- 盈亏统计

### 查看 K 线

在交易记录中点击「查看K线」按钮（需VIP2及以上）：

- 弹窗显示股票 K 线图
- 显示持仓成本线
- 显示止损/止盈线

### 编辑和删除

- 点击编辑按钮修改记录
- 点击删除按钮移除记录
- 删除前会二次确认

## 定时任务

### 功能说明

设置定时执行的任务，如：

- 定时 AI 分析
- 定时发送数据推送

### 管理任务

在「研究」页面选择「定时任务」：

**查看任务列表：**

- 任务名称
- 执行周期 (Cron 表达式)
- 上次执行时间
- 下次执行时间
- 任务状态

**任务操作：**

- 查看详情
- 暂停/恢复任务
- 删除任务

### 股票定时 AI 监控

针对关注的股票设置定时 AI 分析：

1. 在股票详情页面设置
2. 配置 Cron 表达式（如：`0 30 9 * * *` 表示每天9:30）
3. 设置分析模板
4. 系统会在指定时间自动执行 AI 分析

## 设置功能

### 主题设置

- **深色模式/浅色模式**：切换软件主题
- 默认为深色模式

### AI 设置

**启用 AI 功能：**

- 开启后才可使用 AI 分析功能

**配置 AI 模型：**

- 添加多个 AI 配置
- 设置默认模型
- 配置 API 参数

**问题模板：**

- 设置默认的 AI 提问模板

**启用新闻功能：** 开启新闻资讯

### 浏览器设置

## 常见问题

### Q: AI 分析不工作？

A: 检查设置中是否已启用 AI 功能，并正确配置 API Key。

### Q: 数据不更新？

A: 尝试以下方法：

1. 检查网络连接
2. 重启软件
3. 在设置中手动更新基础数据

### Q: 如何导出交易记录？

A: 在交易记录页面，可以查看和编辑记录，数据保存在本地数据库。

### Q: VIP 功能无法使用？

A: 确认赞助码是否正确，输入后需重启软件生效。
