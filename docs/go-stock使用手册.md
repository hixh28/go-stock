## 1. 软件简介

go-stock 是一款基于大语言模型的 AI 赋能股票分析桌面工具，基于 **Wails + Vue3 + NaiveUI** 技术栈构建，支持 A股、港股、美股的行情数据获取与智能分析。

**核心特性：**

- 多市场行情监控（A股/港股/美股）
- AI 智能分析股票（支持多家大模型平台）
- AI 智能体对话（React/PlanExecute 两种模式）
- 实时 K 线图与分时图
- 市场资讯聚合（财联社/新浪/外媒）
- 资金流向与龙虎榜数据
- 筹码分布计算
- 定时任务与自动预警
- MCP 服务与技能扩展
- 弹幕互动社区

**支持的 AI 模型平台：**

DeepSeek、硅基流动、智谱AI、字节豆包、阿里云百炼、Moonshot、腾讯混元、讯飞星火、零一万物、MiniMax、百川智能、百度千帆、OpenAI、Azure OpenAI、OpenRouter、Ollama 以及所有兼容 OpenAI API 格式的平台。

## 2. 安装与启动

### 下载软件

从 GitHub Releases 页面下载对应版本的安装包：

- Windows 版：`go-stock-windows-amd64.exe`
- macOS 版：`go-stock-darwin-universal`

### 首次启动

1. 双击运行程序
2. 首次启动会自动初始化股票基础数据（A 股/港股/美股基础信息）
3. 窗口大小会自适应屏幕分辨率
4. 加载完成后进入主界面

### 数据存储

- 数据库文件：运行目录下 `data/stock.db`（SQLite）
- 字典数据：运行目录下 `data/` 目录
- 配置信息保存在本地数据库中

## 3. 界面布局与导航菜单

### 整体布局

软件采用**底部水平导航栏 + 上方内容区**的布局方式，导航菜单固定在窗口底部。

### 导航菜单结构

| 菜单项 | 图标 | 路由 | 说明 |
|--------|------|------|------|
| ⭐ 股票自选 | StarOutline | `/` | 自选股管理，含分组子菜单 |
| 📰 市场行情 | NewspaperOutline | `/market` | 市场数据与资讯，含12个子菜单 |
| ✨ 基金自选 | SparklesOutline | `/fund` | 基金关注与管理（需配置启用） |
| 🤖 Ai智能体 | Robot | `/agent` | AI智能体对话（需配置启用） |
| 🔬 研究中心 | FlaskOutline | `/research` | 研究分析工具集，含9个子菜单 |
| ⚙️ 设置 | SettingsOutline | `/settings` | 软件配置 |
| 🐙 关于 | LogoGithub | `/about` | 版本信息与赞助 |
| — 隐藏至托盘区 | SlideHide24Filled | — | 最小化到系统托盘 |
| — 退出程序 | PowerOutline | — | 关闭应用 |

> **提示**：「基金自选」和「Ai智能体」两个菜单项需要在设置页面中启用对应功能后才会显示。

### 窗口标题栏

标题栏动态显示当前市场状态，格式为：

```
go-stock A股交易中|港股休市|美股休市  「投资格言」  [数据来源于网络，仅供参考；投资有风险，入市需谨慎]
```

市场状态每 60 秒自动刷新，同时随机切换一条投资格言。

## 4. 股票自选

股票自选是软件的核心页面，用于管理和监控关注的股票。

### 4.1 标签页分组

页面顶部为标签页栏，支持分组管理自选股：

| 操作 | 说明 |
|------|------|
| 点击"全部" | 查看所有自选股 |
| 点击分组标签 | 查看该分组内的股票 |
| 点击"+"按钮 | 新增分组（输入分组名称和排序值） |
| 点击标签关闭按钮 | 删除分组（需二次确认，数据不可恢复） |
| 拖拽标签 | 调整分组排序（"全部"标签不可拖拽） |

### 4.2 自选股卡片

每只自选股以卡片形式展示，包含以下信息区域：

#### 卡片标题区

- **股票代码标签**：显示股票代码
- **取消关注**按钮：移除该股票
- **AI分析**按钮：发起 AI 智能分析（需启用 AI 功能）
- **移出分组**按钮：将股票从当前分组移出（仅在分组标签下显示）

#### 价格信息区

- **当前价格**：实时价格，涨为红色，跌为绿色，平为默认色
- **盘前盘后价格**：仅美股显示盘前盘后价格及涨跌幅
- **涨跌幅百分比**：带数字动画效果
- **今日盈亏金额**：仅在设置了成本价和持仓数量后显示
- **迷你走势线**：仅在分组标签下显示

#### 行情数据区

四宫格显示：
- 最高价 + 最高涨幅百分比
- 最低价 + 最低跌幅百分比
- 昨日收盘价
- 今日开盘价

#### 盘口数据区（仅A股）

可折叠面板，展示买卖五档盘口数据：
- 折叠时显示：买一报价(申报量) / 卖一报价(申报量)
- 展开后显示：买一至买五、卖一至卖五各档的报价和申报量

#### 卡片底部

- 日期和时间
- 持股数量标签（设置持仓后显示）
- 成本盈亏标签（设置成本后显示："成本:xxx*xxx 盈亏% (xxx ¥)"）
- **多周期K线**按钮

#### 卡片操作按钮

| 按钮 | 功能 |
|------|------|
| 成本 | 打开成本设置对话框 |
| 分时 | 打开分时走势图 |
| 日K | 打开日 K 线图 |
| 资金 | 打开资金趋势图（仅A股显示） |
| 详情 | 跳转问财搜索页面 |
| 公告 | 跳转公司公告页面（仅A股） |
| 研报 | 跳转个股研报页面（仅A股） |
| 设置分组 | 下拉菜单，将股票添加到指定分组 |

### 4.3 底部搜索栏

位于页面右下角，包含：

- **自动完成输入框**：输入股票名称/代码/指数关键词，实时模糊匹配
- **关注**按钮：将搜索到的股票添加到自选列表
- **发送弹幕**按钮（需启用弹幕功能）：通过 WebSocket 发送弹幕消息

> **提示**：搜索已关注的股票时，页面会自动滚动到对应卡片并闪烁红色边框5秒。

### 4.4 成本设置对话框

点击"成本"按钮打开，宽度800px，表单字段：

| 字段 | 说明 |
|------|------|
| 股票成本 | 成本价，支持 HK$/¥ 后缀自动适配 |
| 股票数量 | 持仓数量，步进100 |
| 涨跌提醒 | 涨跌幅预警百分比 |
| 股价提醒 | 绝对价格预警 |
| 开仓价 | 开仓价位 |
| 股票排序 | 排序值 |
| 止盈价 | 止盈价格 |
| 止损价 | 止损价格 |

### 4.5 分时图对话框

点击"分时"按钮打开，宽度1000px：

- 使用 ECharts 绘制当日分时走势图
- 包含股价线和成交量柱状图
- 标记最高/最低价位点
- 标记均价线、开盘价线、收盘价线
- 低于开盘价（绿色）、大于开盘价小于收盘价（蓝色）、大于收盘价（红色）分段着色
- **自动刷新**：打开后每 10 秒自动刷新

### 4.6 日K线图对话框

点击"日K"按钮打开，宽度1000px：

- 日 K 蜡烛图
- MA5/MA10/MA20/MA30 均线
- 成交量柱状图
- 数据缩放滑块，默认显示近14%的数据
- 标记最高/最低/平均收盘价
- 鼠标悬停提示框（开盘/收盘/最低/最高/成交量/各均线值）
- 数据来源：近365天K线数据

### 4.7 AI分析对话框

点击"AI分析"按钮打开，宽度800px：

#### 内容展示区

- Markdown 格式渲染 AI 分析结果
- 加载状态旋转动画
- 导出 PDF 工具栏

#### 底部信息区

- 左：模型名称标签 + 分析时间
- 右：风险提示 "*AI分析结果仅供参考，请以实际行情为准。投资需谨慎，风险自担。"

#### 操作区域

- **工具调用开关**：开启/关闭 AI 函数工具调用
- **思考模式开关**：开启/关闭深度思考模式
- **三个下拉选择框**：
  - AI 模型服务配置
  - 系统提示词
  - 用户提示词
- **问题输入框**：自定义提问内容
- **操作按钮**：

| 按钮 | 功能 |
|------|------|
| 开始AI分析 | 重新发起 AI 流式分析 |
| 保存为图片 | 截图保存为 PNG |
| 复制到剪切板 | 复制分析结果文本 |
| 保存为Markdown | 保存为 `.md` 文件 |
| 保存为Word | 导出为 `.docx` 文件 |
| 分享到项目社区 | 分享到社区 |

### 4.8 资金趋势对话框

点击"资金"按钮打开，宽度1000px：

- 使用 MoneyTrend 组件展示近 360 天资金趋势图
- 图表高度 500px

### 4.9 多周期K线对话框

点击"多周期K线"按钮打开，最大宽度 1100px：

- 使用轻量级 K 线图组件（TradingView Lightweight Charts）
- 支持拖拽调整开仓价/止损价/止盈价/成本价线，调整后自动保存
- **VIP权限控制**：VIP2 及以上用户可正常使用，权限不足时提示并自动关闭
- 美股代码（`gb_` 前缀）不支持多周期K线

### 4.10 价位线预警系统

每次价格更新时自动检查：

| 预警类型 | 触发条件 | 推送类型 |
|----------|----------|----------|
| 涨跌幅预警 | 涨跌幅达到设定百分比 | 类型1 |
| 股价预警 | 股价触及设定价格 | 类型2 |
| 成本价预警 | 成本盈亏达到设定值 | 类型3 |
| 止盈触及 | 当前价 ≥ 止盈价 | 类型4 |
| 止损触及 | 当前价 ≤ 止损价 | 类型5 |

> 防抖机制：同一价格不重复提醒，两次通知间隔至少60秒。

### 4.11 弹幕系统

- 连接 WebSocket 服务器实时接收弹幕
- 使用 vue3-danmaku 组件展示弹幕飘过效果
- 可在设置中开启/关闭弹幕功能

## 6. 基金自选

基金自选页面用于关注和管理基金产品。

> **注意**：此功能需在设置页面中启用"基金自选"后，菜单才会显示。

### 6.1 基金卡片

- **涨幅标签组**：近一月/三月/六月/一年/三年/五年/今年来/成立来

- **操作按钮**：

| 按钮 | 功能 |
|------|------|

### 6.2 底部搜索栏

- 自动完成输入框：输入基金名称/代码搜索
- "关注"按钮：将基金添加到关注列表
- "发送弹幕"按钮：发送弹幕消息

每 60 秒自动刷新关注列表数据。

## 7. AI智能体

AI 智能体页面提供对话式交互体验，支持两种 Agent 模式。

> **注意**：此功能需在设置页面中启用"AI智能体"后，菜单才会显示。

### 7.1 界面布局

- **聊天区域**：主体消息列表，支持滚动
- **执行步骤区域**：当 AI 使用工具时，显示步骤列表和状态指示（已完成=绿色、工具调用=蓝色、活跃=橙色）
- **推理过程区域**：展开/收起 AI 的思考过程
- **分析报告区域**：可折叠的"📊 分析报告"区域，AI 生成的结构化报告

### 常见问题

#### Q: AI 分析不工作？

A: 检查以下几项：
1. 设置中是否已启用"AI诊股总开关"
2. 是否正确配置了 AI 模型的 API Key
3. 网络连接是否正常
4. 如使用代理，确认代理设置正确

#### Q: 数据不更新？

A: 尝试以下方法：
1. 检查网络连接
2. 确认当前是否为交易时间（休市期间部分数据不会刷新）
3. 重启软件
4. 在设置中启用"启动时更新基础信息"

#### Q: 如何添加自选股？

A: 在股票自选页面右下角的搜索栏中输入股票名称或代码，选中后点击"关注"按钮。

#### Q: VIP 功能无法使用？

A: 确认以下几项：
1. 赞助码是否正确输入
2. 点击"验证"按钮确认赞助码有效
3. 重启软件使 VIP 状态生效

#### Q: 如何收到股票预警通知？

A: 需要完成以下配置：
1. 在成本设置中设置涨跌提醒百分比或股价提醒
2. 在设置中开启"本地推送"或配置"钉钉推送"
3. 确保在交易时间内运行软件

#### Q: AI 智能体菜单不显示？

A: AI 智能体和基金自选菜单需要在设置页面中启用对应功能后才会显示。前往设置页面，检查相关开关是否开启。

#### Q: 多周期K线提示权限不足？

A: 多周期K线功能需要 VIP2 及以上等级。请在关于页面查看当前 VIP 等级，并在设置中输入赞助码升级。
